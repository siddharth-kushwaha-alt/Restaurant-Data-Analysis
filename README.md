# 🍽️ Restaurant Data Analysis Using Excel

## 📌 Project Overview

This project was completed as part of my **Data Analysis Internship at Cognifyz Technologies**.

The objective of this project was to analyze a restaurant dataset using Microsoft Excel and extract meaningful business insights through data analysis, Pivot Tables, formulas, and visualizations.

---

## 🎯 Objectives

The project focuses on answering business-related questions such as:

- Identify the top 3 most common cuisines.
- Analyze restaurant distribution across different cities.
- Calculate average restaurant ratings by city.
- Visualize price range distribution.
- Determine the percentage of restaurants offering online delivery.
- Compare ratings of restaurants with and without online delivery.
- Analyze restaurant ratings and voting patterns.
- Identify popular cuisine combinations.
- Perform geographical analysis using location data.
- Analyze restaurant chains and their popularity.

## 📈 Key Analysis Performed

### ✅ Top Cuisines Analysis
- Identified the three most common cuisines.
- Calculated their percentage distribution.

### ✅ City Analysis
- Determined the city with the highest number of restaurants.
- Calculated average ratings for each city.
- Identified the city with the highest average rating.

### ✅ Price Range Analysis
- Visualized the distribution of restaurant price ranges.
- Calculated the percentage of restaurants in each price category.

### ✅ Online Delivery Analysis
- Determined the percentage of restaurants providing online delivery.
- Compared ratings between restaurants with and without online delivery.

### ✅ Restaurant Ratings Analysis
- Analyzed rating distribution.
- Calculated the average number of customer votes.

### ✅ Cuisine Combination Analysis
- Identified popular cuisine combinations.
- Compared their average ratings.

### ✅ Geographic Analysis
- Explored restaurant locations using geographic data.
- Identified regional patterns.

### ✅ Restaurant Chain Analysis
- Identified restaurant chains.
- Compared their ratings and popularity.
